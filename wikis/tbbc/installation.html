Mod Installation

Mod installation varies on how and where your Minecraft instance is installed.
Some of these options assume you have the mod jar file.

  • Default installation in MC Launcher (needs jar file):
    First, download my mod from one of the sources above.
    Then, hold down the windows key + r and type %appdata%. Proceed to the .minecraft folder,
    and then go to the mods folder. Drop the jar file for the mod into the mods folder. Done!

  • CurseForge app installation:
    You should be able to click add more content on the modpack profile and search for the mod.
    If that fails, download the mod jar file from one of the sources above.
    After downloaded, go to your modpack profile in File Explorer.
    You should be able to find the profile's folder by going to the profile's settings and clicking Open Folder.
    From there, just drag the mod's jar file into the mods folder.

  • MultiMC installation (needs jar file):
    Find your MC instances folder in MultiMC through the Folders button at the top.
    In File Manager, find the instance you want, double click it and then double click the .minecraft folder,
    and drag the mod jar file into the mods folder.
